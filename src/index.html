APSS

The aim is to create a system that allows the detection and visualization of air pollution of various kinds (electromagnetic, CO, etc.). A system of data acquisition and transmission, to be installed on a Drone, which makes measurements, mapping any area of interest, and transmits them to a system of data analysis / post-processing that displays the levels of pollution through flat maps or colored volumetric. The need is to provide an efficient measurement tool for the fight against pollution, which can become a means of awareness if you think of the use of augmented reality techniques to visualize the level maps contextualized in the real environment.
